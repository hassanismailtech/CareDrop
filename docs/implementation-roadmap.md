# CareDrop implementation roadmap

## Phase 0 — foundation
- Create the monorepo structure and shared tooling.
- Establish design tokens and core layout primitives.
- Set up authentication shell and routing.

## Phase 1 — patient experience
- Build the home and search experience.
- Implement product listing, detail, cart, and checkout.
- Add prescription upload flow and order confirmation.

## Phase 2 — pharmacy and admin
- Implement pharmacy dashboard for order review and inventory updates.
- Create admin views for user and order oversight.

## Phase 3 — rider and logistics
- Add rider order assignment and map-based delivery view.
- Connect real-time status updates and delivery proof.

## Phase 4 — integrations and hardening
- Integrate payments, messaging, and mapping services.
- Add analytics, monitoring, and offline-safe behaviour.
